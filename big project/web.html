Projet: Web

But du projet

Le but de mon premier projet était de créer un site en HTML et CSS sur le jeu de mon choix. Pour le deuxième, j'ai dû concevoir un site sur les différents projets que j'ai réalisés tout au long de l'année en utilisant du JavaScript et le framework Materialize. Pour mon troisième et dernier projet, j'ai dû reprendre le deuxième site et le transformer en un site PHP avec MySQL.

Contraintes

Les contraintes que j'avais pour ces trois projets étaient de créer des sites qui soient responsives, notamment grâce à Materialize. Mes sites devaient être optimisés pour le référencement naturel (SEO). En PHP, je devais intégrer un formulaire d'inscription et de connexion qui serait géré par un rôle administrateur et un panneau d'administration pour gérer les utilisateurs et les pages directement sur le site. Enfin, mes sites devaient être protégés contre les injections SQL et chaque page devait être accessible uniquement par les utilisateurs autorisés.
