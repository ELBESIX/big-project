Projet: Python

But du projet

Nous avons eu deux projets en Python. Le premier consistait à créer deux jeux, le pierre-papier-ciseaux et le morpion. Le deuxième consistait à écrire un code pour effectuer du scraping sur un site web.

Contraintes

Les contraintes du premier projet étaient de rédiger du pseudo-code avant de commencer à coder les jeux, afin d'apprendre à en créer et de faciliter leur codage par la suite. Pour le deuxième projet, nous devions écrire un code pour effectuer du scraping sur un site web. Nous avions pour objectif de récupérer des informations et de les stocker dans un tableau.
